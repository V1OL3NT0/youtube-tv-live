# YouTube TV Live

YouTube TV Live is an open source Android TV application that creates a
linear TV-like experience using public YouTube channels.

## Features
- Android TV support
- Continuous playback (TV mode)
- Live streams and long videos
- Official YouTube player
- Simple channel configuration
- Open source (MIT)

## Default channels
- @ancap_su
- @ancapsuclassic
- @visao_libertaria
- @ideiasradicais

## Legal
This project does not download, redistribute, or rehost YouTube content.
All playback is done using the official YouTube player.
